# Example Configuration Files

This folder contains example configuration files for setting up your local environment.

## dj_local_conf.example.json
Copy this file to your secrets location (e.g., `dj_local_conf.json`) and fill in your database credentials and paths. Do **not** commit your real credentials.

## google_creds.example.json
This is a template for a Google service account credentials file. Download your real credentials from the Google Cloud Console and save as `google_creds.json` in a secure location. Do **not** commit your real credentials.
